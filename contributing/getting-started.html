Getting started

Note that you can also contribute without writing OCaml code!

This guide will get you making your first PR to the Dark repo. It helps you set up the repo, write your first test, and make your first pull request. After that, it will guide you on how to work on bigger things.

Before you do that, it's helpful to:

  • Join the #contributors channel in the Dark Slack (join using this invite).
  • Sign the CLA
